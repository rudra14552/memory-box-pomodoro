# Memory Box — Photo Keepsake + Study Timer

A little handmade desk gadget built around an SSD1306 128×64 OLED display — part digital keepsake, part study companion:

- 💌 **Memory slideshow** — cycles through 6 special reference images, each with its own meaningful date stamped in the corner, plus a soft camera-flash LED effect while it idles
- 📚 **Study timer (Pomodoro-style)** — adjustable in 5-minute steps, started with a button hold, for focused study/work sessions
- 🔔 **Session alert** — buzzer + flashing LED for 10 seconds when a study session ends

Everything runs non-blocking (no `delay()` in the main logic), so button presses always feel instant and responsive.

## Hardware

| Component | Pin | Notes |
|---|---|---|
| Arduino Nano | — | Main microcontroller |
| SSD1306 OLED (128×64, I2C) | SDA/SCL | I2C address `0x3C` |
| Button 1 (mode select) | D3 | Enter/exit study timer |
| Button 2 (adjust/start) | D2 | +5 min / hold to start session |
| Buzzer | D13 | Session-end tone (2 kHz) |
| LED | D12 | Idle flash effect / session-end flash |
| Slide switch | — | Power on/off |
| 18650 Li-ion battery | — | Powers the whole unit |
| TP4056 (or similar) charging/power bank module | — | Charges the 18650 and regulates power to the Nano |

Buttons are wired with `INPUT_PULLUP`, so connect each button between its pin and GND — no external resistors needed.

## Libraries required

Install via the Arduino Library Manager:

- `Adafruit GFX Library`
- `Adafruit SSD1306`

## How it works

The sketch is built as a simple state machine with three modes:

### 1. Slideshow mode (default)
Displays one of 6 special images, each with its own date label in the bottom-right corner. The active image auto-advances every **59 minutes**, so it keeps rotating gently through the memories over time. While idle, the LED runs a soft strobe pattern (flash–pause–flash–longer pause) like a camera going off.

### 2. Study timer mode
Press **D3** to enter. Then:
- Tap **D2** to add 5 minutes to the session length (wraps back to 5 after 60)
- Hold **D2** for 800ms+ to start the countdown
- Press **D3** again at any time to cancel and return to the slideshow

### 3. Session-end alert
Triggered automatically when the study session reaches zero. Buzzer and LED pulse together for 10 seconds — time for a break — then the device returns to the slideshow automatically.

## Swapping in your own photos & dates

Each image is a `128×64` monochrome bitmap stored in `PROGMEM` as a byte array (1 bit per pixel, 1024 bytes each). To load your own special images and dates:

1. Convert a photo to a 128×64 monochrome bitmap using a tool like [image2cpp](https://javl.github.io/image2cpp/) (settings: *Draw mode: horizontal, 1 bit per pixel*). Simple, high-contrast images (line art, silhouettes, cropped portraits) tend to look best at this resolution.
2. Paste the generated byte array in place of one of the existing `bitmapN[]` arrays.
3. Update the matching entry in the `dates[]` array with the date or label you want shown next to that image (anniversary, first date, inside joke — whatever it means to the two of you).

```cpp
const unsigned char* bitmaps[] = {
  bitmap1, bitmap2, bitmap3, bitmap4, bitmap5, bitmap6
};
String dates[] = {
  "13/09/25", "20/09/25", "06/02/26",
  "15/02/26", "08/03/26", "18/03/26"
};
```

To change how long each memory stays on screen before advancing, edit:

```cpp
const unsigned long SCREEN_INTERVAL = 59UL * 60UL * 1000UL; // 59 minutes
```
